# Security Policy

## 报告问题

请不要在公开 Issue 中提交 API Key、访问令牌、真实文档内容或可复现的用户数据。安全问题应通过仓库维护者的私密联系方式报告，并包含受影响版本、复现步骤和影响范围。

## 公开演示边界

- Sites Worker 是公开流量的唯一入口。
- Vercel FastAPI 的受保护接口要求共享代理密钥。
- 公开文档必须列入 `PUBLIC_DOCUMENT_IDS` 白名单。
- 公开上传与索引变更保持关闭；批量问答状态按匿名访客隔离。
- `.env`、生产数据、上传文件和运行日志不得提交到 Git。

密钥疑似泄露时，应立即轮换并重新部署前后端。
